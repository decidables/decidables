Model Prediction: Predicting Performance for Hyperbolic Temporal Discounting

From model parameters to predicted performance

If we have model parameters, either from fitting human data, or derived from theoretical considerations, we may want to simulate task performance in order to generate predictions. In other words, we can have our model perform the task as if it were a participant.
